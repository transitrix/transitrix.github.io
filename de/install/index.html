Installieren

Transitrix Studio rendert die Notationen live in Ihrem Editor. Wählen Sie Ihren Kanal — die Erweiterung ist überall identisch.

Transitrix Studio — in Ihrem Editor

  • VS Code — der VS-Code-Marketplace. Oder suchen Sie Transitrix Studio im Erweiterungen-Panel.
  • Cursor, VSCodium, Windsurf — die Open-VSX-Registry. Dieselbe Erweiterung und Kennung; suchen Sie Transitrix Studio im Erweiterungen-Panel.
  • IntelliJ IDEA — der JetBrains-Marketplace. Öffnen Sie Einstellungen → Plugins → Marketplace und suchen Sie Transitrix Studio. Installiert auch in die anderen JetBrains-IDEs.

Oder starten Sie mit dem Claude-Code-Skill

Lieber Schritt für Schritt geführt? Der Onboarding-Skill legt ein Repository an — oder orientiert Sie in einem, das Ihre Organisation bereits hat — und erstellt Ihre erste Datei. In einer Claude-Code-Sitzung:

/plugin marketplace add transitrix/methodology
/plugin install transitrix@transitrix-methodology
/transitrix:onboard

Empfohlene Ergänzung

Installieren Sie neben Studio eine Mermaid-Vorschau-Erweiterung — Markdown Preview Mermaid Support (im VS-Code-Marketplace und auf Open VSX) — für standardmäßige Mermaid-Diagramme: Flussdiagramme, Sequenz, ER, Gantt.

Kein Lock-in. Diagramme-als-Text sind größtenteils standardmäßiges Mermaid, und Transitrix baut auf offenen Standards auf — ArchiMate 3.2 und BPMN 2.0. Studio ergänzt nur die Enterprise-Notationen, die Mermaid nicht abdeckt. Ihre Dateien bleiben reiner, diff-barer Text auf einer offenen, MIT-lizenzierten Methodik — lesbar mit oder ohne unsere Werkzeuge.
